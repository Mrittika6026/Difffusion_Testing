# Diffusion Models

Implementation of Diffusion and Stable Diffusion models in PyTorch.

## Models

- **Diffusion Model**: Trained on Flowers102 dataset
- **Stable Diffusion**: Trained on Pokemon Images dataset

## Datasets

- Flowers102
- Pokemon: [Kaggle Dataset](https://www.kaggle.com/datasets/vishalsubbiah/pokemon-images-and-types)

## Usage

Open and run the Jupyter notebooks:
- `diffusion.ipynb` - Diffusion model training
- `stable_diffusion.ipynb` - Stable diffusion model training
